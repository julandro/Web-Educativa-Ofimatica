¿Qué es PowerPoint?

Microsoft PowerPoint es un software de presentación visual que forma parte de la suite Microsoft Office. Se utiliza para crear presentaciones de diapositivas, que pueden incluir texto, imágenes, gráficos, animaciones, videos y otros elementos multimedia.
